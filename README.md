# Sistema-IoT-monitoreo-de-gases-
Diseñar e implementar un sistema IoT de monitoreo georreferenciado de la exposición ocupacional a gases (CH₄, CO₂, H₂S) y material particulado (PM2.5, PM10) para operarios de recolección de residuos, con visualización geoespacial de la telemetría mediante un geoportal web.
